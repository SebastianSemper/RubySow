Every feature was included with a certain intention but still with as mush flexibility as possible. Keep on reading to find out more what to expect from RubySow.

  1. Lists

    2. Populate and maintain lists

    We also need to organize similar content in one place since that is the same thing we do in our mind. We think of You can populate them manually, search in directories with regular expressions or generate them by applying functions to other lists. Finally you can wrap the content of a list with web code and dump it into the HTML file of your website. Wherever and however you like.

  2. Patching

    3. Parse and insert another file

    The maybe most important thing it to split, things up, into manageable parts. There is nothing worse than browsing through a massive amount of code that most of the time repeats itself a lot since well - that's how websites are. One page often contains several instances of the same thing like images, tables or whole pieces of text.

  3. Trees

    4. Create and recursively work with them

    While normally being a concept in math and information technology trees can come in very handy while working with nested structures like nested lists, file hierarchies or the structure of a document. You can generate trees by iterating through directories or by hand. Afterwards you can apply a function to the nodes and in the end just drop everything into your web document.
